# LayaAir-TypeScript-Automation-Demo
自动化开发
1 保存后自动编译
2 编译完成后自动刷新浏览器
# instation
安装nodejs
全局安装 gulp
    npm install -g gulp

初始化项目
    npm init

安装开发依赖
    npm install --save-dev gulp gulp-connect

在项目根目录下创建一个名为 gulpfile.js 的文件

这是示例项目的源码 https://github.com/wildfirecode/LayaAir-TypeScript-Automation-Demo
